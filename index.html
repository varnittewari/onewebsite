Project Move

projectMove is a personal project I started to work on in the Summer of 2014.

projectMove is a personal project I started to work on in the Summer of 2014. This project was a continuation of the famous sixthsense project conducted by MIT engineer Pranav Mistry. The renowned TED Talks speaker Mistry released open source information regarding his project and I decided to continue to work on it by contributing to the code and developing the traits of the project. projectMove, my spin-off of the original project entails a gesture-controlled environment where the user can utilize any flat platform as a screen and control a number of applications with gestures of their hands.

Additionally, this pendant-like wearable device has audio and vibration reactions for the blind and visual support for the deaf. To learn more about the original work as well as my spin-off Click Here.

Super Microwave

Circuitry built to represent an efficient microwave

I developed the Super Microwave project as an extension of an introduction to digital systems course project. The project required us to implement an entire microwave system using and FPGA and its relative components. After the project was completed and submitted, I spent some time developing it further by making the system accessible to people with visual difficulty. I integrated an audio-vibration system that indicates different signals for different operations as in one vibrate and buzz sound for one minute, two for two minutes and so forth.
